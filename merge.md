## git merge

**git merge *existing_branch_name***

Объединить две ветки можно параметром **merge** с указанием имени ветки. Команда объединит указанную ветку с основной.

**git merge *origin***

Слияние удалённого репозитория с локальным выполняется параметром **merge** с указанием имени удалённого репозитория - например, *origin*.

Команда:

```bash=
git merge
```
